Chapter 2 â€“ Getting Started with HTML

HTML (HyperText Markup Language) is the skeleton of a web page. Every web page you see online is built on HTML. Tags like <h1>, <p>, <img> define content.

Basic HTML Structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage!</p>
  </body>
</html>

The <!DOCTYPE html> tells the browser this is an HTML5 document. The <head> contains metadata like title and links. The <body> contains the visible page content.

Chapter 4 â€“ Introduction to CSS

CSS (Cascading Style Sheets) makes websites beautiful. You can change colors, fonts, layouts, and animations.

Example CSS:

h1 {
  color: blue;
  text-align: center;
}
p {
  font-size: 18px;
  color: gray;
}

Ways to Add CSS:

  1. Inline: Inside HTML tags using style="".
  2. Internal: Inside <style> in the <head>.
  3. External: A separate style.css file linked in HTML.

Chapter 5 â€“ CSS Layout with Flexbox

Flexbox is a CSS layout model that makes arranging items simple. Example: 

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

With flexbox you can easily build navigation bars, responsive grids, and align content vertically.
